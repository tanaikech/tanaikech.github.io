Summary

From Fig. 1, it was found the following point.

  • The tag triggers is used a lot at 2013. At that time, this tag is used for 2075 questions. After 2013, this frequency of triggers is linearly reduced. The average frequency of use for the questions is 1229.

From Table 1, it was found the following points.

  • triggers is used with sql-server, sql, mysql and oracle. These are shown in the tag info. Ref
  • Languages of jquery, php, c#, javascript, python, Java, html, node.js and so on are used with triggers tag.
  • When the tags including google are retrieved, it was found that google-apps-script, google-sheets, google-form, google-cloud-functions, google-cloud-functions, google-apps-script-simple-triggers, google-cloud-firestore, google-tag-manager, google-cloud-platform, google-docs and google-apps-script-editor were used with triggers tag.

From Fig. 2, it was found the following point.

  • Questions with google-apps-script and triggers are increased from 2012.

From these situation, in the current stage, I would like to propose to use the event trigger of Google Apps Script as google-apps-script and triggers.
