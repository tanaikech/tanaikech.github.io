09 May 2017, 09:36
ShTips
bash / dos

Input JSON

text.txt :

{
  "values": [
    [
      "a1",
      "b1",
      "c1"
    ],
    [
      "a2",
      "b2",
      "c2"
    ],
    [

      "b3",
      "c3"
    ],

      "a4",

      "c4"
    ],

      "b5",
      "c5"

Windows DOS

type text.txt | jq -c "."

Linux Bash

$ cat text.txt | jq -c '.'
