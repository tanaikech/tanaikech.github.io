Gemini CLI: Featuring an Enhanced ToolsForMCPServer

09 Jul 2025, 15:22
GasTips
Google Apps Script / Gemini / Model Context Protocol / MCP

Gists

Abstract

This report introduces ToolsForMCPServer, an enhanced Google Apps Script library that expands the capabilities of Gemini CLI. It showcases new tools that streamline complex workflows, with a special emphasis on facilitating seamless file content transfer and management between a user’s local environment and Google Drive.

Introduction

This report details significant enhancements to ToolsForMCPServer, a powerful Google Apps Script library designed to work in tandem with Gemini CLI. By integrating this library with a Model Context Protocol (MCP) server, the capabilities of Gemini CLI are dramatically expanded, especially in its interaction with Google Workspace services. This document will explore the core architecture that makes this possible, introduce the new tools available in the library, and demonstrate their power through practical examples that bridge the local command line with the cloud.

The Core Architecture: How It Works

The power of this system lies in the synergy between three key components:

  1. Gemini CLI: An open-source tool that brings the power of the Gemini model directly to your command-line interface.
  2. MCP Server: A Model Context Protocol (MCP) server built using Google Apps Script Web Apps. This server acts as the brain, equipped with tools that can interact with Google Workspace services like Drive, Gmail, and Sheets.
  3. ggsrun CLI tool: A command-line utility that acts as a bridge, enabling the crucial transfer of file content between your local machine and the MCP Server running on Google Apps Script.

Together, these components create a powerful feedback loop where you can issue natural language commands in your terminal, and Gemini CLI can orchestrate actions that involve both your local files (via ggsrun) and cloud services (via the MCP Server). This architecture is the foundation for seamless file manipulation between your computer and Google Drive.

The development of this enhanced toolset builds upon a series of foundational reports:

  • Building Model Context Protocol (MCP) Server with Google Apps Script
  • Gemini CLI with MCP Server: Expanding Possibilities with Google Apps Script
  • Gemini CLI with MCP Server Built by Web Apps of Google Apps Script
  • Processing File Content Using Gemini CLI with an MCP Server Built by Google Apps Script

The Enhanced Toolkit: ToolsForMCPServer

To make this ecosystem even more powerful, the ToolsForMCPServer Google Apps Script library was published. The core idea is simple: the more tools the MCP server has, the more you can achieve. This latest update adds several new tools to the library, further expanding the realm of possibilities.

Newly Added Tools

In this update, the following 9 tools were added to the library:

  • get_attachment_files_from_Gmail: Retrieves attachment file IDs from a Gmail message.
  • rename_files_on_google_drive: Renames files on Google Drive.
  • move_files_on_google_drive: Moves files or folders to a different location on Google Drive.
  • convert_mimetype_of_file_on_google_drive: Changes the MIME type of files on Google Drive.
  • generate_roadmap_to_google_sheets: Creates a roadmap in Google Sheets.
  • generate_description_on_google_drive: Generates and sets a description for a file on Google Drive.
  • generate_image_on_google_drive: Generates an image from a prompt and saves it to Google Drive.
  • summarize_file_on_google_drive: Summarizes a file stored on Google Drive.
  • description_web_site: Provides descriptions of websites given their URLs.

In the current stage (July 9, 2025), 32 tools are provided by ToolsForMCPServer for the MCP server.

Summary

  • Powerful Integration: By combining Gemini CLI with a Google Apps Script-based MCP Server, you can unlock powerful new capabilities that extend beyond the command line into the Google Workspace ecosystem.
  • Seamless File Transfer: The ggsrun CLI tool is a critical component, acting as a bridge that enables smooth uploading and downloading of files between your local machine and Google Drive, orchestrated by natural language prompts.
  • Enhanced Toolkit: The ToolsForMCPServer library significantly expands the range of actions you can perform, with new tools for renaming, moving, describing, and generating content directly on Google Drive.
  • Complex Workflow Automation: This system allows Gemini CLI to orchestrate multi-step workflows, such as uploading a local file, processing it in the cloud (e.g., summarizing a PDF or generating a presentation), and then sending the result via email or downloading it back to your machine.
  • Extensible and Open: The architecture is fundamentally extensible. Developers can add their own custom tools to the Google Apps Script library, creating near-infinite possibilities for custom automation and integration.
